# Weather Application

Welcome to the Weather Application! This application provides real-time weather information with a dynamic background based on current weather conditions.

## Features

- Responsive and modern design.
- Real-time temperature, weather condition, humidity, and wind speed display.
- Dynamic background images based on weather conditions.
- User-friendly interface with city selection.

## Project Overview

This project utilizes the WeatherAPI to fetch real-time weather data. The application dynamically updates its background image and displays key weather information for the selected city.

## How to Use

1. Open the `index.js` file and replace the `API_KEY` variable with your WeatherAPI key.

```javascript
const key = 'YOUR_WEATHERAPI_KEY';
```

2. Open the `index.html` file in your web browser.
